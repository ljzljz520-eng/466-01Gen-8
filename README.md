# 📷 工业相机标定任务台

工业相机内参标定系统，支持工程师创建产线、镜头、标定板和拍摄距离后，系统引导上传多角度棋盘格照片并自动计算相机内参结果。标定失败时会明确指出错误原因：**照片模糊**、**角点不足**还是**参数不一致**。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🏭 **产线管理** | 创建、删除产线，管理生产线信息 |
| 📷 **镜头管理** | 创建、删除镜头，记录镜头型号、焦距等参数 |
| ⬛ **标定板管理** | 创建棋盘格标定板，配置行列数和方格尺寸 |
| 📋 **标定任务** | 创建标定任务，关联产线、镜头、标定板和拍摄距离 |
| 🎯 **拍摄引导** | 提供9种拍摄角度引导（正面、左偏、右偏、上仰、下俯等） |
| 📤 **照片上传** | 支持拖拽和点击上传，实时检测照片质量 |
| 🔍 **质量检测** | 自动检测照片模糊度和角点数量，标记无效照片 |
| 🧮 **内参计算** | 使用OpenCV张正友标定法计算相机内参矩阵和畸变系数 |
| ❗ **错误诊断** | 标定失败时明确指出原因：照片模糊、角点不足或参数不一致 |
| 📊 **结果展示** | 显示内参（fx, fy, cx, cy）、畸变系数（k1, k2, p1, p2, k3）、重投影误差 |

## 📖 使用指南

### 第一步：创建产线

1. 点击顶部导航栏「产线管理」
2. 点击「+ 新建产线」
3. 填写产线名称（如：`SMT产线A`）和描述
4. 点击「创建产线」

### 第二步：创建镜头

1. 点击顶部导航栏「镜头管理」
2. 点击「+ 新建镜头」
3. 填写：
   - 镜头名称（如：`A01工位相机`）
   - 所属产线（选择上一步创建的产线）
   - 镜头型号（如：`Computar M1214-MP2`）
   - 焦距（如：`12` mm）
4. 点击「创建镜头」

### 第三步：创建标定板

1. 点击顶部导航栏「标定板管理」
2. 点击「+ 新建标定板」
3. 填写：
   - 标定板名称（如：`7x9棋盘格25mm`）
   - 内角点行数：棋盘格横向内角点数（如：`6`）
   - 内角点列数：棋盘格纵向内角点数（如：`8`）
   - 方格尺寸：单个方格的实际边长（如：`25` mm）
4. 点击「创建标定板」

> 💡 **提示**：内角点 = 棋盘格数 - 1。例如 8×10 格的棋盘格，内角点为 7×9。

### 第四步：创建标定任务

1. 点击顶部导航栏「标定任务」
2. 点击「+ 新建标定任务」
3. 填写：
   - 任务名称（如：`A01相机标定任务`）
   - 选择镜头
   - 选择标定板
   - 拍摄距离（相机到标定板的距离，单位 mm）
4. 点击「创建任务」

### 第五步：上传标定照片

1. 在任务列表中点击「查看详情」
2. 在「拍摄引导」区域选择当前上传照片的拍摄角度
3. 点击或拖拽照片到上传区域
4. 系统会自动检测每张照片的：
   - **清晰度得分**（拉普拉斯方差，>100为合格）
   - **角点数量**（应等于标定板的内角点总数）
5. 绿色边框 = 有效照片，红色边框 = 无效照片
6. 建议上传 **5~15张** 不同角度的有效照片

> 📐 **推荐拍摄角度**：正面、左偏15°、右偏15°、上仰15°、下俯15°、左前、右前、左后、右后

### 第六步：开始标定计算

1. 确保至少有 **3张有效照片**（推荐5张以上）
2. 点击底部「🚀 开始计算内参」按钮
3. 等待几秒钟，系统会自动：
   - 对所有照片进行角点提取
   - 检查照片一致性
   - 运行张正友标定算法
   - 计算内参矩阵和畸变系数
4. 标定成功后会显示：
   - ✅ 重投影误差（< 1.0像素为优秀）
   - 🧮 内参矩阵（fx, fy, cx, cy）
   - 🔧 畸变系数（k1, k2, p1, p2, k3）

## ❗ 错误诊断说明

系统会对标定失败给出明确的错误原因：

| 错误类型 | 原因说明 | 解决方案 |
|---------|---------|---------|
| **照片模糊** | 照片清晰度不足，拉普拉斯方差 < 100 | 重新拍摄，确保对焦准确、光照充足 |
| **角点不足** | 未检测到棋盘格角点或数量不够 | 确保标定板完整出现在画面中，光线均匀 |
| **参数不一致** | 各照片角点数量不一致或拍摄角度过于相似 | 删除异常照片，补充更多不同角度的照片 |
| **标定计算失败** | 算法无法收敛或重投影误差过大 | 检查标定板参数是否正确，替换低质量照片 |
| **照片数量不足** | 有效照片少于3张 | 继续上传更多有效照片 |

## 🔬 核心算法说明

### 1. 模糊检测算法 ([calibration.py#L30-L36](file:///Users/kkcarrot/solo/466/466-01Gen-8/app/calibration.py#L30-L36))

使用 **拉普拉斯方差（Variance of Laplacian）** 方法：
- 对灰度图应用 3×3 拉普拉斯核
- 计算输出图像的方差
- 方差越大 → 图像越清晰
- 阈值：`BLUR_THRESHOLD = 100.0`

```python
laplacian_var = cv2.Laplacian(gray_img, cv2.CV_64F).var()
```

### 2. 角点检测算法 ([calibration.py#L38-L56](file:///Users/kkcarrot/solo/466/466-01Gen-8/app/calibration.py#L38-L56))

使用 OpenCV 的棋盘格角点检测：
- `cv2.findChessboardCorners()` 快速检测
- `cv2.cornerSubPix()` 亚像素级精度优化
- 返回所有内角点的亚像素坐标

### 3. 相机标定算法 ([calibration.py#L115-L230](file:///Users/kkcarrot/solo/466/466-01Gen-8/app/calibration.py#L115-L230))

使用 **张正友标定法**（Zhang's Method）：

**输出参数：**

| 参数 | 含义 |
|------|------|
| fx, fy | 焦距（像素单位） |
| cx, cy | 主点坐标（像素单位，通常在图像中心附近） |
| k1, k2, k3 | 径向畸变系数 |
| p1, p2 | 切向畸变系数 |

**质量评估：**
- **重投影误差（Reprojection Error）**：将3D点投影回2D图像与实际检测点的平均距离
- 单位：像素
- < 0.5 px：非常优秀
- < 1.0 px：合格
- \> 1.0 px：需要重新拍摄

## 💡 使用建议

### 拍摄技巧

1. **光照均匀**：避免强烈反光或阴影
2. **标定板完整**：确保棋盘格完整出现在画面中
3. **多角度拍摄**：正面、左右倾斜、上下倾斜、前后倾斜各拍几张
4. **对焦清晰**：确保棋盘格线条锐利，无明显模糊
5. **照片数量**：推荐 5~15 张有效照片

### 提高标定精度

1. 使用打印质量好的标定板（建议贴在平板上防止弯曲）
2. 拍摄距离与实际工作距离一致
3. 标定板覆盖画面的不同区域（中心、边缘、四角）
4. 避免使用严重畸变的鱼眼镜头照片（算法不支持极端畸变）

## 🐛 常见问题

**Q: 为什么所有照片都显示"角点不足"？**
A: 请检查标定板参数（行列数）是否与实际标定板一致，确保标定板完整出现在画面中。

**Q: 为什么一直提示"参数不一致"？**
A: 请删除检测异常的照片，补充更多不同角度的有效照片。确保所有照片使用的是同一个标定板。

**Q: 重投影误差太大怎么办？**
A: 检查拍摄时光照是否充足、对焦是否准确，删除低质量照片后重新标定。
